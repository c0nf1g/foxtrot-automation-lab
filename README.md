# foxtrot-automation-lab
Automation tests for https://www.foxtrot.com.ua/

# Setting Up Development Environment

Create a virtualenv:

```
$ python3 -m venv venv
```

Activate the virtualenv:

```
$ source venv/bin/activate
```

Install requirements:

```
$ pip install -r requirements.txt
```

Run tests:

```
pytest
```
